## ğŸ”‘ Key Components

### 1. Clerk (External Service)
- Handles user signup/signin
- Issues JWT tokens
- Manages user sessions
- Email verification
- Password resets

### 2. Backend Authentication Layer (`/lib/auth.ts`)
- Verifies JWT tokens
- Fetches user data from Clerk
- JIT user provisioning
- Role management
- Middleware functions

### 3. Database (`users` table)
- Stores application-specific user data
- Links Clerk identity (via `clerkId`)
- Manages roles and permissions
- Tracks creation/update timestamps

### 4. Protected Endpoints
- Use `requireAuth()` for authenticated-only access
- Use `requireRole()` for role-based access
- Access user data via `req.user`
